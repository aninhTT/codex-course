# Lesson 6.04: Final Refinement Lab

## Learn

The course should end with a small but useful system. This lesson reviews the starter skill, custom workflow skill, and prototype together and asks Codex for the smallest improvements that would make them more useful in real work.

## Practice

Try this in Codex:

```text
Help me do a final refinement pass on my course artifacts. Review my starter skill, my custom workflow skill, and my local prototype, then recommend the smallest set of changes that would make each one more useful in my real workflow.
```

**Success criteria:** The learner leaves with a practical refinement list and a clearer sense of which artifact they want to keep improving first.
