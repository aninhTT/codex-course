# Lesson 6.06: Final Refinement Lab

## Learn

The course should end with a small but useful system. This lesson reviews the starter skill, custom workflow skill, prototype, and any Goal ideas together. The point is to ask Codex for the smallest improvements that would make the full system more useful in real work.

## Practice

Try this in Codex:

```text
Help me do a final refinement pass on my course artifacts. Review my starter skill, my custom workflow skill, my local prototype, and any Goal ideas I drafted. Then recommend the smallest set of changes that would make each one more useful in my real workflow.
```

**Success criteria:** The learner leaves with a practical refinement list and a clearer sense of which artifact or Goal idea they want to keep improving first.
