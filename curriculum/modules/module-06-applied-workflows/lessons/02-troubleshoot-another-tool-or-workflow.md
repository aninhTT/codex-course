# Lesson 6.02: Troubleshoot Another Tool or Workflow

## Learn

One of the most valuable intermediate Codex habits is using it to troubleshoot surrounding tools and workflows. The real value comes from structured diagnosis, not quick guessing.

## Practice

Try this in Codex:

```text
Help me troubleshoot a workflow or tool issue related to this course work. Start by asking what is going wrong, what I expected, and what I already tried. Then suggest a structured debugging path.
```

**Success criteria:** Codex narrows the problem through questions first, and the learner gets a structured path they can actually follow.
