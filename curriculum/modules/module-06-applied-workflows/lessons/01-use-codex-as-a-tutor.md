# Lesson 6.01: Use Codex as a Tutor

## Learn

Codex is not only useful for doing work. It is also useful for helping the learner think through work. This lesson normalizes asking Codex to teach, clarify, and guide rather than immediately take over.

This is a standing best practice, not just a final-module move. When you feel stuck, ask Codex to slow down, recap what it sees, and explain the next step before doing more work.

## Practice

Try this in Codex:

```text
Act as my Codex tutor. Based on the artifacts in this course workspace, ask what I'm stuck on first, recap where I am, then explain the next best step in plain language instead of doing everything for me right away.
```

**Success criteria:** Codex asks clarifying questions first and helps the learner understand the next step instead of racing straight into execution.
