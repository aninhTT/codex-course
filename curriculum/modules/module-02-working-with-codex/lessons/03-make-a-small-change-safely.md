# Lesson 2.03: Make a Small Change Safely

## Learn

This lesson is the first transition from read-only help to a real edit. The task is intentionally narrow so the learner can focus on the edit-and-review loop rather than on complexity.

## Practice

Try this in Codex:

```text
Rewrite `practice-workspace/notes/weekly-review-draft.md` into a cleaner weekly recap using `practice-workspace/templates/weekly-recap-template.md`. Keep the tone practical and concise, then tell me what changed.
```

**Success criteria:** Codex makes a focused improvement to the file, explains what changed, and the learner reviews the result before accepting it as complete.
