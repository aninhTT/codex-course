# Lesson 2.05: Create A File You Can Find

## Learn

One of the most useful beginner moments is seeing Codex create a real file that you can find on your computer. This makes Codex feel less abstract. It is not just chatting; it is working with files.

After Codex creates something, go find it and read it yourself. That is how you stay oriented and build trust.

## Practice

Try this in Codex:

```text
Create a new file in `practice-workspace/notes` called `my-codex-course-notes.md`. Fill it with a short, clear summary of what I've learned so far in Modules 1 and 2, and leave a section at the bottom for notes I want to keep adding later.

After you create it, tell me exactly where you saved it, remind me to open it myself, and summarize what I should review.
```

**Success criteria:** Codex creates the file in the correct folder, the learner can find it, and the learner reviews the file directly.
