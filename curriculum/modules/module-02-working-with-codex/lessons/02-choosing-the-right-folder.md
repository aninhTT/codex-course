# Lesson 2.02: Choosing The Right Folder

## Learn

Folder choice affects what Codex can inspect and change. A broad folder gives Codex more context. A smaller folder keeps the work focused.

Use this course workspace when you are taking lessons. Use `practice-workspace/` for guided exercises. Use `playground/` for safe experiments. For a real skill, prototype, or work project, you may eventually open a separate Codex project so Codex can focus on that folder.

If you are unsure, ask Codex: "Is this the right project for this task?"

## Practice

Try this in Codex:

```text
Look at this course workspace and help me understand the folder choices. Explain when I should use the whole course project, `practice-workspace/`, `playground/`, a smaller folder, a separate Codex project, or no project at all. Give me one simple rule of thumb before I ask Codex to edit anything.
```

**Success criteria:** The learner can name a broad-folder use case, a smaller-folder use case, and a no-project use case.
