# Lesson 2.04: Make A Small Change Safely

## Learn

This is the first real edit. Keep it narrow so the learner can focus on what changed and how to review it.

Best practice: ask for the smallest useful change, then ask Codex to summarize what changed afterward. That gives you something real to inspect without turning the exercise into a mystery.

## Practice

Try this in Codex:

```text
Rewrite `practice-workspace/notes/weekly-review-draft.md` into a cleaner weekly recap using `practice-workspace/templates/weekly-recap-template.md`. Keep the tone practical and concise. Make the smallest useful change, then tell me what changed and what I should review.
```

**Success criteria:** Codex makes a focused improvement, explains what changed, and the learner reviews the result before treating it as complete.
