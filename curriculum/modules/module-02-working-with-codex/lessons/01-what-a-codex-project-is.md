# Lesson 2.01: What A Codex Project Is

## Learn

A Codex project is basically a local folder Codex can work inside. The project gives Codex context: files, folders, notes, apps, drafts, and anything else stored there.

You do not always need a project. If you are asking a general question, brainstorming, or drafting something from scratch, a plain conversation may be enough. But when you want Codex to inspect files, edit files, build a skill, or prototype something, the project matters.

Beginner mental model: **the project is the box Codex can look inside.** Choose the box that contains the work you want help with.

## Practice

Try this in Codex:

```text
Explain what a Codex project is using this course folder as the example. Tell me what Codex can see in this project, why that context helps, and when I might use Codex without opening a specific project or folder.
```

**Success criteria:** The learner can explain that a Codex project is a folder of local context and that some conversations do not require project files.
