# Lesson 2.02: Review Before You Trust

## Learn

Codex can be fast and useful, but the learner is still responsible for the final result. Reviewing a proposal before accepting an edit is one of the most important habits in the course.

## Practice

Try this in Codex:

```text
Take `practice-workspace/notes/weekly-review-draft.md` and propose a cleaner version using `practice-workspace/templates/weekly-recap-template.md`, but do not edit the file. Show me what you would change and why.
```

**Success criteria:** Codex proposes a better structure without editing the file, and the learner can compare the suggestion against the original before moving on.
