# Module 2: Working With Codex Day to Day

This module is where learners start getting their hands dirty in a good way. The focus is not on advanced building yet. It is on learning how to work with Codex in a steady, confident way.

## What You'll Learn

- how to separate explain, plan, and act
- how to review before accepting edits
- how to let Codex make a small safe change
- how to have Codex create a real file the learner can go find
- how to spot beginner-friendly day-to-day use cases
