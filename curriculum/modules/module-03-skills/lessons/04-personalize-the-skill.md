# Lesson 3.04: Personalize the Skill

## Learn

This lesson shifts from scaffolding to ownership. The learner is not trying to expand the skill wildly. They are making it a little more useful for their real work without losing focus.

Best practice: personalize by clarifying the recurring job first, then making the smallest useful change. If the skill starts trying to solve three different jobs, split the ideas apart.

## Practice

Try this in Codex:

```text
Take my `weekly-recap-helper` skill and help me personalize it for my real work. Ask me 3 short questions first, then suggest the smallest changes needed to make the skill more useful without making it overly complex. Before editing, tell me what you plan to change and why.
```

**Success criteria:** Codex asks clarifying questions first, suggests restrained improvements, and the learner can name the recurring task the skill now supports best.
