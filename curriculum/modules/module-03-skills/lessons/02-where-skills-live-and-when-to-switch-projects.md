# Lesson 3.02: Where Skills Live And When To Switch Projects

## Learn

Skill work may not always belong inside the course project. This course can teach the concept and hold practice materials, but a real skill might live in a separate folder or Codex project.

Before switching projects, ask Codex to summarize what you did and where to return. After switching back, paste or reference that summary so Codex can help you resume.

Project check:

- Am I in the course project, `playground/`, or a separate project?
- What files should Codex be able to see?
- What should Codex summarize before I switch away?
- What lesson should I return to after the skill work?

## Practice

Try this in Codex:

```text
Help me decide where skill work should happen for this course. Explain when to use the course workspace, when to use `playground/`, and when to create or open a separate Codex project for a real skill. Then give me a short "before I switch projects" prompt and a short "I'm back, help me resume" prompt.
```

**Success criteria:** The learner understands that skill work can happen in another project and has prompts for switching away and returning.
