# Lesson 3.03: Inspect A Skill Before Building

## Learn

Before building a first skill, inspect one. The learner does not need to understand every detail. They just need to see that a skill is structured instructions plus any supporting resources that help Codex do a focused job.

In this course, `skill-creator` is treated as a Codex environment skill, not something that lives inside the course folder itself. If it is available, use it. If it is not available, Codex should explain skill structure using another local skill or a simple example.

## Practice

Try this in Codex:

```text
If the `skill-creator` skill is available in this Codex environment, show it to me and walk me through its structure like a beginner. If it is not available, use another local skill or a simple example instead. Focus on what parts matter most if I want to build my own simple skill.
```

**Success criteria:** Codex explains the main parts of a skill and the learner sees what matters for a simple first build.
