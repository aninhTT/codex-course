# Lesson 3.01: What a Skill Is

## Learn

Skills are one of the clearest ways to move from one-off prompting to repeatable workflows. A skill gives Codex a reusable set of instructions and, when needed, supporting resources for one kind of task.

The key beginner idea is simple: if you keep asking Codex to do the same kind of work, a skill may be the right next step.

Example: a skill could be a reusable workflow for comparing two Excel sheets and outputting the same kind of new sheet every time. That is more structured and repeatable than rewriting the same prompt over and over.

## Practice

Try this in Codex:

```text
Explain skills to me like I'm new to Codex. Use the files in `reference/glossary.md` and this course workspace if helpful. Focus on what skills are, when they help, and how they differ from a normal prompt or something like a GPT in ChatGPT.
```

**Success criteria:** Codex explains skills as reusable workflows, contrasts them with one-off prompts, and keeps the explanation practical for a beginner.
