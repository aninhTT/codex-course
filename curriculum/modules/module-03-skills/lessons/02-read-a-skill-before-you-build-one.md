# Lesson 3.02: Read a Skill Before You Build One

## Learn

Before building a first skill, it helps to inspect a real one. The learner does not need to understand every detail. They just need to see how a skill is structured and what parts matter most.

In this course, `skill-creator` is treated as a Codex environment skill, not something that lives inside the course folder itself. If it is available in the learner's Codex setup, use it. If it is not available, Codex should still explain skill structure using another local skill or a simple course example.

## Practice

Try this in Codex:

```text
If the `skill-creator` skill is available in this Codex environment, show it to me and walk me through its structure like a beginner. If it is not available, use another local skill or a simple course example instead. Focus on what parts matter most if I want to build my own simple skill.
```

**Success criteria:** Codex identifies the main parts of the skill structure, explains them clearly, and helps the learner see what matters for a simple first skill.
