# Lesson 3.06: Build, Test, And Refine A Custom Workflow Skill

## Learn

Skills are easier to build well when Codex plans before it writes anything. Use the workflow summary from Lesson 3.05, review the plan, build the first version, test it with realistic scenarios, and make one useful iteration.

If the skill belongs in a separate project, summarize before switching away and summarize again before returning to the course.

## Practice

Step 1: type this into Codex:

```text
Let's plan this custom skill before building it. Use the workflow summary from the previous lesson as the starting point. Outline the goal, trigger, inputs, steps, output, edge cases, and what is out of scope. Do not build anything yet. I want to review the plan first.
```

Step 2: after the plan looks right, type this:

```text
Now build the first version of this custom skill. If it should live in another project, help me summarize before switching. Keep it focused on the workflow we planned. After building, help me test it with 2 realistic scenarios, identify what is weak, and make one useful refinement.
```

**Success criteria:** The learner reviews the plan, builds or clearly locates the skill, tests it with realistic examples, and makes one focused iteration.
