# Lesson 3.04: Build A Guided First Skill

## Learn

The first skill should solve one narrow, useful problem. In this course, the guided skill is a recap helper so the exercise stays grounded in the practice files.

Guided skill:

- Name: `weekly-recap-helper`
- Job: turn rough notes into a cleaner weekly recap with priorities, wins, blockers, and next steps
- Starter sources: `practice-workspace/notes/weekly-review-draft.md` and `practice-workspace/templates/weekly-recap-template.md`

If your Codex setup wants skill files in a separate project, pause and use the project-switching prompts from Lesson 3.02 before leaving the course project.

## Practice

Try this in Codex:

```text
Help me create a beginner Codex skill named `weekly-recap-helper`. It should turn rough weekly notes into a cleaner recap using the template in this workspace. Keep the skill lean, practical, and easy for a beginner to understand. If this work should happen in another project or folder, tell me before building and give me a summary I can use to return to this lesson. After creating it, summarize what files were created or changed and how I should inspect the skill.
```

**Success criteria:** The learner ends with a simple, focused skill draft or a clear project-switching plan if the skill should live elsewhere.
