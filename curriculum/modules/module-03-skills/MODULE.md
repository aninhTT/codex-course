# Module 3: Skills, Explained and Built

This module is where things start getting especially fun. The learner moves from one-off prompting into reusable Codex workflows and starts seeing how a skill can save real time.

## What You'll Learn

- what a skill is and when it helps
- how to inspect an existing skill without getting overwhelmed
- how to build a simple first skill
- how to personalize a skill for real work
- how to use a stronger example skill to generate ideas for what to build next
