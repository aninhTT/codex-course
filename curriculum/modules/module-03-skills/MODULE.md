# Module 3: Skills And Repeatable Workflow Automation

This module is where learners move from one-off prompts into repeatable Codex workflows. It combines the old skills module and workflow-skill module into one clearer arc: understand skills, know where skill work should happen, build a guided skill, then build one custom workflow skill.

## What You'll Learn

- how prompts, repeatable workflows, and skills differ
- when skill work should happen in this course project, `playground/`, or a separate project
- how to inspect an existing skill without getting overwhelmed
- how to build a simple first skill
- how to choose and interview a workflow before building
- how to build, test, and refine a custom workflow skill
