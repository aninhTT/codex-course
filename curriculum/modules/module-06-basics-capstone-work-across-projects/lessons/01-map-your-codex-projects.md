# Lesson 6.01: Map Your Codex Projects

## Learn

By now, the learner may have more than one place where Codex work happens:

- the course project
- `practice-workspace/`
- `playground/`
- a skill project
- a prototype project
- a future real-work project

Mapping projects helps the learner know where to ask Codex to work and what context Codex should be able to see.

## Practice

Try this in Codex:

```text
Help me map my Codex projects. List what belongs in the course project, `practice-workspace/`, `playground/`, a skill project, a prototype project, and a future real-work project. For each one, tell me what Codex should be able to see and what should stay out of that project.
```

**Success criteria:** The learner has a simple map of project types and understands which work belongs where.
