# Lesson 6.04: Foundation Handoff To Next-Level Modules

## Learn

The basics path ends by checking what the learner has built and what is ready for the next-level modules.

Foundation artifacts may include:

- a notes file from Module 2
- a guided skill and a custom workflow skill from Module 3
- a prototype plan or small app from Module 4
- a Goal draft from Module 5
- a project map from Module 6

Modules 7-10 build on these artifacts with recorded workflows, Sites, safe recurrence, and a power-user capstone.

## Practice

Try this in Codex:

```text
Help me do a foundation handoff before I start Module 7. Review my course artifacts so far: notes file, guided skill, custom workflow skill, prototype or prototype plan, Goal draft, and project map. Tell me what is ready, what is missing, and which artifact would be best to use in Modules 7-10.
```

**Success criteria:** The learner knows which foundation artifacts are ready and what to carry into the next-level modules.
