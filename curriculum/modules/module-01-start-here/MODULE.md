# Module 1: Start Here in Codex

Welcome to the foundation of the course. This module gives learners the beginner mental models they need before they start building more ambitious things in Codex.

## What You'll Learn

- what Codex is in practical terms
- how Codex differs from chat-only AI tools
- why local folders and workspace context matter
- how local coding tools work with files that live on your own computer
- how to complete a first safe task using sample files created for this course
