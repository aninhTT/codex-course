# Module 1: What Codex Is

Welcome to the foundation of the course. This module gives learners the simplest useful mental model for Codex before they start changing files or building artifacts.

## What You'll Learn

- what Codex is in practical terms
- how Codex differs from chat-only AI tools without getting lost in product comparisons
- how Codex can chat, inspect files, edit files, build artifacts, and use tools
- how to complete a first read-only task using sample files created for this course
