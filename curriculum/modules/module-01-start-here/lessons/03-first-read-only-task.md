# Lesson 1.03: First Read-Only Task

## Learn

The first Codex task should be safe and concrete. Before asking Codex to change anything, ask it to inspect a real file and explain what it sees.

This course includes sample practice files inside `practice-workspace/`, so the learner can try file-based Codex work without bringing their own notes yet.

Read-only tasks are useful because they build trust. Codex can show that it understands the file before you let it edit.

## Practice

Try this in Codex:

```text
This course includes sample practice files for me. Open `practice-workspace/notes/meeting-notes-2026-03-03.md` and tell me what kind of file it is and what it seems to contain. Then summarize it into 5 bullets and list the likely action items. Do not edit any files.
```

**Success criteria:** The learner sees Codex inspect and summarize a real file accurately, and no files are edited during the exercise.
