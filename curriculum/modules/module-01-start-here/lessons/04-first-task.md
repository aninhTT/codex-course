# Lesson 1.04: Open the Practice Workspace and Do a First Task

## Learn

The course should move quickly from explanation to action. To make that possible, this course includes sample files for practice inside `practice-workspace/`. This first task uses one of those sample files so the learner does not need to bring their own notes yet.

## Practice

Try this in Codex:

```text
This course includes sample practice files for me. Start by opening `practice-workspace/notes/meeting-notes-2026-03-03.md` and telling me what kind of file it is and what it seems to contain. Then summarize it into 5 bullets and list the likely action items. Do not edit any files yet.
```

**Success criteria:** The learner understands that the course created sample files for practice, Codex summarizes the note accurately, and no files are edited during the exercise.
