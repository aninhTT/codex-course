# Lesson 5.02: Guided Dashboard Build

## Learn

The goal of this lesson is to make a first meaningful product change without turning the course into a full frontend class. Codex should help the learner improve the dashboard while keeping the app small and intentional.

## Practice

Try this in Codex:

```text
Turn this starter app into a cleaner personal work dashboard. Keep it simple and local. Add one more section for "This Week's Focus", improve the wording, and keep the design intentional rather than generic.
```

**Success criteria:** The app gains a meaningful improvement in structure or content, and the changes feel focused rather than overbuilt.
