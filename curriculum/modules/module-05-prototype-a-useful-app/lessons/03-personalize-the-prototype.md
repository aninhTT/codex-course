# Lesson 5.03: Personalize the Prototype

## Learn

The prototype becomes more valuable when it is adapted to one real workflow. This lesson asks Codex to clarify first, then make the lightest useful changes so the prototype still feels manageable.

## Practice

Try this in Codex:

```text
Help me adapt this dashboard to one of my real use cases from Module 4. Ask me 3 short questions first, then update the app so it supports that workflow in a simple way.
```

**Success criteria:** The learner ends with a prototype that supports one specific job, not a generic dashboard or an overgrown app.
