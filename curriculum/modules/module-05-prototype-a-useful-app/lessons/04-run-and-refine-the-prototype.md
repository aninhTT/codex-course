# Lesson 5.04: Run and Refine the Prototype

## Learn

This lesson closes the loop. The learner should end with a runnable local artifact and a short refinement list rather than an unfinished mockup.

Best practice for prototypes: preview early. It is easier to improve a small thing you can see than a big idea you are only imagining.

## Practice

Try this in Codex:

```text
Help me verify this local app works. Show me the simplest way to preview it, then review the app for clarity and suggest 3 small refinements that would improve usefulness without making it more complex. Do not expand the scope yet.
```

**Success criteria:** The learner can preview the app locally and identify a small, practical next round of improvements.
