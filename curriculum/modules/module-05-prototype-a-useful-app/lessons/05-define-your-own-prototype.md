# Lesson 5.05: Define Your Own Prototype

## Learn

By this point, the learner has seen how to work from a starter app. Now the goal is to choose a prototype idea they actually want to make, ideally based on the use cases they generated earlier in the course.

## Practice

Try this in Codex:

```text
Help me define a prototype I actually want to build based on the use cases I generated earlier in the course. Ask me a few short questions, then help me choose one idea that is realistic for a first version.
```

**Success criteria:** The learner ends the lesson with one clearly defined prototype idea that feels small enough to start and useful enough to matter.
