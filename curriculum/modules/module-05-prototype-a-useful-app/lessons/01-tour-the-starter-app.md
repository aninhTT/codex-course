# Lesson 5.01: Tour the Starter App

## Learn

Product-building feels much less intimidating when the learner sees the pieces first. This lesson makes the path explicit: the learner will start with a guided starter app, learn how it works, improve it, and then later in the module move into planning and building one of their own use cases.

## Practice

Try this in Codex:

```text
Walk me through the starter app in `practice-workspace/starter-app/` like I'm a beginner. Explain what each file does and how the HTML, CSS, and JavaScript work together.
```

**Success criteria:** Codex explains the structure clearly, and the learner can point to which file controls structure, style, and content.
