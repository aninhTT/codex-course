# Module 5: Prototype a Useful App in Codex

This module is a chance to make something that feels a little more alive. Codex helps the learner start from a guided prototype, then branch into a version of their own.

## What You'll Learn

- how to inspect a small app with Codex
- how to make a first meaningful product change
- how to personalize a starter prototype for a real workflow
- how to define and plan your own small prototype
- how to build and iterate on that prototype without overcomplicating it
