# Module 5: Goals And Longer Codex Work

By now the learner has worked with files, skills, and prototypes. This module introduces Goal mode for longer work with a clear finish line and evidence source.

## What You'll Learn

- when a normal prompt is enough
- when a Goal is useful
- what makes a strong Goal
- how project context affects Goal quality
- how to draft a bounded, evidence-backed Goal
