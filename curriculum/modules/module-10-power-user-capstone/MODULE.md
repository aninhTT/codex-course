# Module 10: Power-User Capstone

You already built separate artifacts; now you'll map them into one practical workflow system.

This is not another polish pass like Module 6. It is a systems view. The learner connects a workflow skill, a recorded-workflow idea or skill, a prototype or Sites plan, and an automation design card into one practical operating map.

## What You'll Learn

- how to decide which Codex surface should handle each part of a workflow
- how to combine course artifacts without making the system too complicated
- how to review risks, missing evidence, privacy concerns, and next refinements
- how to leave the course with a realistic operating plan
