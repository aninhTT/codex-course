# Lesson 10.01: Map Your Workflow System

## Learn

The capstone connects what you built without turning everything into one giant tool. A workflow system map shows which artifact does which job.

A simple map might include:

- a custom workflow skill for repeatable instructions
- a recorded workflow skill or recording plan for steps that are easier to show
- a prototype or Sites plan for a visual workflow surface
- an automation design card for safe recurrence
- a Goal idea for longer evidence-backed work

The point is clarity. If an artifact does not help the workflow, leave it out.

## Practice

Try this in Codex:

```text
Help me create a workflow system map from my course artifacts. Include my custom workflow skill, recorded workflow idea or skill, prototype or Sites plan, automation design card, and any Goal idea that still matters. For each one, say what job it does and what should stay out of scope.
```

**Success criteria:** The learner has a system map that assigns a clear job to each useful artifact.
