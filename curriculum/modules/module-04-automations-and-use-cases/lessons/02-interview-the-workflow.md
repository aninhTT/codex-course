# Lesson 4.02: Interview the Workflow

## Learn

Before Codex can help build a good skill, it needs a clear picture of the workflow. This means understanding the trigger, the inputs, the steps, the output, and where the workflow tends to break down.

## Practice

Try this in Codex:

```text
Interview me so we can define this workflow clearly enough to turn it into a custom skill. Ask me about the trigger, inputs, steps, edge cases, and desired output until you have a clear description of what the skill should do.

At the end, give me a concise workflow summary that you can carry into the next lesson when we plan the skill.
```

**Success criteria:** Codex interviews the learner step by step, and the resulting workflow description is detailed enough to build from.
