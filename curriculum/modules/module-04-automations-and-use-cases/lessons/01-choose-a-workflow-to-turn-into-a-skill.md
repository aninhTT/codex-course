# Lesson 4.01: Choose a Workflow to Turn Into a Skill

## Learn

Not every use case is the right first custom skill. The best beginner choice is a workflow that is repetitive, clear enough to describe, and small enough to test quickly.

## Practice

Try this in Codex:

```text
Based on the use cases I generated in Module 3, help me choose one workflow to turn into a custom skill. Compare my top options and recommend the most beginner-friendly one to build first.
```

**Success criteria:** The learner ends the lesson with one clear workflow chosen for a custom skill, and the choice feels realistic for a first build.
