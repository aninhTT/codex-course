# Lesson 4.04: Test and Refine the Skill

## Learn

Skills are rarely perfect on the first try. The point of this lesson is to normalize testing, noticing weak spots, and iterating instead of expecting the first version to be done.

## Practice

Try this in Codex:

```text
Help me test this custom skill with 2 or 3 realistic scenarios. Show me what is working well, what is still weak, and what the next refinements should be. Then help me make one useful iteration on the skill.
```

**Success criteria:** The learner runs realistic tests, identifies at least one meaningful improvement, and makes at least one iteration on the custom skill.
