# Module 8: Sites And Hosted Prototypes

You already built a local prototype; now you'll learn the review and publishing layer.

This module does not rebuild the prototype curriculum. Instead, it teaches the questions a learner should answer before asking Codex to save or deploy a hosted site: what should be reviewed, who should see it, whether it needs storage, and what must stay out of the source files.

## What You'll Learn

- how local preview, saved versions, and deployments differ
- how to decide whether a prototype is ready for Sites
- how to think about access, secrets, environment values, and storage
- why deployment is optional and should be intentional
- how to ask Codex for a Sites-readiness review
