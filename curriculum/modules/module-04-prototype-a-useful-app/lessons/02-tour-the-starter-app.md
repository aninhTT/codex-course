# Lesson 4.02: Tour The Starter App

## Learn

Product-building feels less intimidating when the learner sees the pieces first. This lesson uses the starter app in `practice-workspace/starter-app/` to show how a simple local app is organized.

The learner does not need to become a frontend engineer. They only need the basic map: structure, style, and behavior.

## Practice

Try this in Codex:

```text
Walk me through the starter app in `practice-workspace/starter-app/` like I'm a beginner. Explain what each file does and how the HTML, CSS, and JavaScript work together. Do not change any files yet.
```

**Success criteria:** Codex explains the app structure clearly, and the learner can point to which file controls structure, style, and behavior.
