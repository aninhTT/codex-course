# Lesson 4.04: Project Check: Course Workspace, Playground, Or New Project

## Learn

Before building your own prototype, decide where it belongs.

Use the course workspace when you are following the guided lesson. Use `playground/` when you want a small experiment that can live inside the course. Use a separate Codex project when the prototype is becoming a real project with its own files, assets, or future work.

Before switching away, ask Codex to summarize the current lesson, what changed, and where to return. When you come back, ask Codex to resume from that summary.

## Practice

Try this in Codex:

```text
Help me decide where my prototype should live: this course workspace, `playground/`, or a separate Codex project. Ask what I want to build, how real it is, and whether it needs its own folder. Then give me a short summary prompt to use before switching projects and a return prompt for when I come back to the course.
```

**Success criteria:** The learner chooses a project location and has clear prompts for switching away and returning.
