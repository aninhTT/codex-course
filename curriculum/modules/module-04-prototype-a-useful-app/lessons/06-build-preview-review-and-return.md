# Lesson 4.06: Build, Preview, Review, And Return

## Learn

The prototype build should follow the approved v1 plan. After building, preview it, review it, and summarize before returning to the course project if you worked elsewhere.

This lesson reinforces the project habit: when you leave the course workspace for real work, bring back a summary so Codex can help you continue the course.

## Practice

Step 1: type this into Codex:

```text
Build the first version of this prototype based on the v1 plan we approved. Keep the scope tight. After building, tell me how to preview it and what I should review first.
```

Step 2: after previewing, type this:

```text
Review this prototype with me. Tell me what is working, what feels weak, and one useful iteration we could make later. Then summarize what we built, where it lives, and what lesson I should return to in the course.
```

**Success criteria:** The learner builds or starts the prototype in the chosen project, previews it, reviews it, and leaves with a return summary.
