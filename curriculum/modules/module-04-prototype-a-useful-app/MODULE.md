# Module 4: Prototype A Useful App

This module turns the learner from workflow instructions into a small working app. It keeps the build beginner-friendly while making project-switching explicit: some prototypes can stay in `playground/`, while others deserve their own Codex project.

## What You'll Learn

- what prototypes are for
- how a local app is structured
- how to make one guided prototype change
- when to use the course workspace, `playground/`, or a separate project
- how to define, build, preview, review, and return from a prototype project
