# Use-Case Skill Summary

Source repo: [aninhTT/skill-bank](https://github.com/aninhTT/skill-bank)  
Guided skill for this course: `use-case`

## Why This Skill Is Included

This skill is a useful teaching example because it follows a real workflow instead of answering a one-off question. It gathers context, finds pain points, groups them into themes, and turns them into concrete AI opportunities.

## What The Skill Does

The `use-case` skill is designed to find automation or AI opportunities from workplace signals such as meeting notes, Slack threads, and email. It asks for role and team context first, then looks for pain points, repeated work, blockers, and underused data.

## What Learners Should Notice

- It gathers the right context before making suggestions.
- It breaks a complex job into phases.
- It uses a repeatable evaluation approach.
- It turns messy signals into practical recommendations.

## How This Course Uses It

Learners use this skill to:

1. install a real GitHub-hosted skill
2. inspect how a stronger skill is structured
3. borrow its thinking process
4. brainstorm 2 to 3 Codex use cases for their own work
