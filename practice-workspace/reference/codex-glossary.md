# Codex Glossary

## Workspace

The folder Codex is currently looking at. What Codex can inspect and change depends on this context.

## Skill

A reusable set of instructions and resources that help Codex perform a specific kind of job consistently.

## Automation

A recurring Codex task that runs on a schedule.

## Prompt

The instruction you give Codex. Good prompts are specific about goal, files, and output.

## Review

The step where you inspect what Codex produced before accepting it.
